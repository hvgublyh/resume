原因 : 返回的函数中的i已经变成了4,因此在函数执行时返回值是16
注意 : 返回函数不要引用任何循环变量，或者后续会发生变化的变量。

**解决办法 : 新建函数并立即执行将i的值封闭到函数内**

}

})(i));

var f2 = results\[1\]; //4
var f3 = results\[2\]; //9
