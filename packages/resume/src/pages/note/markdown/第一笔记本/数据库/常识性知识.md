2、关系型数据库
以行和列的形式来存储数据，行和列组成了表，表和表之间存在着一定的关系
3、常用的数据库

4、术语
table：表：以行和列的形式组织数据
col:列：称为字段，在表里体现为表头
row ：行：称为一条记录
