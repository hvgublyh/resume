建立vue-cli项目
2019年11月1日
8:12

1.安装node.js
node -v 以及 npm -v 检查是否安装完成
2.安装webpack
npm install webpack -g

webpack -v 检查是否安装完成
3.搭建vue-cli手脚架
npm install @vue/cli -g

vue -V(注:大写的V)检查是否安装完成
4.新建项目文件夹
d:/VUE
6.进入该文件夹
cd /d d:/VUE
7.建立项目vue_test
vue init webpack vue_test
